Architecture

A modular pipeline: three independent modality streams → fusion core → generative summarizer. Each stream runs a pretrained/fine-tuned model and returns a confidence-scored emotion label.

Language Stream
DistilRoBERTa
Text / transcript analysis
Visual Stream
DeepFace

Notes

  • • Modular design lets each stream be updated independently.
  • • Fusion tolerates missing/noisy modalities by leveraging consensus.
  • • Responsible AI layer filters toxic content and flags bias before generation.

Ethics & Responsible AI

Affective systems can cause harm if left unchecked. This layer ensures the system is designed to be safe, fair, and explainable.

Fairness
Bias detection & mitigation
Dataset checks, adversarial debiasing, demographic balancing.
Safety
Toxicity & threat filtering
Detoxify + rules to block dangerous outputs and escalate if required.
Transparency
Interpretability & logs
Confidence scores, per-modality contributions, and summary rationales.
Responsible AI Checklist
  • • Ensure diverse training data and document dataset provenance.
  • • Provide per-prediction confidence and contributor breakdown.
  • • Avoid high-stakes decisions without human oversight.
  • • Maintain logs for auditing and continual bias monitoring.
